# FOV Changer for Modern Warfare 2 (2009) Campaign

Changes FOV for Call of Duty: Modern Warfare 2 (2009) Campaign

To change your FOV with this FOV Changer,
- Run `mw2fov.exe`
- Type your desired FOV
